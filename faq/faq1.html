1. "I want to learn programming, but I don’t know what to make."
{

   If you’re stuck on what to build, try combining programming with your
   other interests or hobbies.
 
   Programming can be a powerful tool to explore and enhance things you already love:
   
   Like music? Build a playlist organizer.
   Into fitness? Create a workout tracker.
   Enjoy gaming? Make a small game or a custom tool for a game you play.
   
   Using programming alongside your passions is a great way to stay motivated
   while learning, and you’ll get practical use out of the things you build!
}
